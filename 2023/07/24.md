- Node.jsをMySQLに接続するには、mysqlパッケージを利用する mpm install mysql
- mysqlパッケージを読み込み、createConnectionメソッドで、データベースに接続するための情報を定数に代入する
```javascript:test.js
const mysql = require('mysql');

const connection = mysql.createConnection({
  host: 'localhost',
  user: 'progate',
  password: 'password',
  database: 'list_app'
});
```
